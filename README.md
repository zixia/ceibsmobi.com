# ceibsmobi.com

CEIBS Alumni Mobile Club Website

![CEIBS Alumni Mobile Club Website](docs/images/ceibsmobi-website.webp)

## Usage

```sh
docker run \
    --rm -ti \
    -e CEIBSMOBI_MYSQL_HOST \
    -e CEIBSMOBI_MYSQL_USER \
    -e CEIBSMOBI_MYSQL_PASS \
    -p 8080:80 \
    ghcr.io/zixia/ceibsmobi.com
```
