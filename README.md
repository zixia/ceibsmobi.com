# ceibsmobi.com

CEIBS Alumni Mobile Club Website

## Usage

```sh
docker run \
    --rm -ti \
    -e MYSQL_HOST \
    -e MYSQL_USER \
    -e MYSQL_PASS \
    -e MYSQL_DATABASE \
    -p 8888:80 \
    ceibsmobi
```
